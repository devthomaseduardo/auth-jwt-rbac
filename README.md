# Sistema de Autenticação JWT e Permissões

API backend desenvolvida com Node.js, TypeScript, Fastify, Prisma e PostgreSQL.

## Objetivo

Criar um sistema de autenticação real com cadastro, login, senha criptografada, JWT, refresh token e controle de permissões por roles.

## Stack

- Node.js
- TypeScript
- Fastify
- Prisma
- PostgreSQL
- JWT
- bcrypt
- Zod

## Funcionalidades planejadas

- Cadastro de usuários
- Login
- Hash de senha
- Access token JWT
- Refresh token
- Middleware de autenticação
- Controle de acesso por roles
- Rotas públicas e privadas
- Validação de dados
- Documentação com exemplos em curl
